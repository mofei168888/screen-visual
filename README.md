## 大屏可视化

- React.js
- Echarts.js
- Node.js
- Socket.io
- Wow.js

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev

# node server
npm run server

# build for production with minification
npm run build
```
